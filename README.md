# Simple NodeJS Service API with defense via JWT

## Clone this repository locally
`git clone https://github.com/jsmuster/ang6-jwt-login-node-api ng-api`

`cd ng-api`

`npm install`

## Development server

Start debug mode via 

`node --inspect ./bin/www`

Or

`npm start`
